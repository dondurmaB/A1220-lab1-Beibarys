# Overview
This program takes as input a directory of receipt images, and for every receipt outputs the receipt date, the total amount spent, the vendor name, and a category chosen from a predefined list, by calling a language model (GPT-4.1-mini).

# Getting started

## Prerequisites
This program needs pip, python and openai library.

## Installation
Install openai library using 
```pip install openai```.

# Usage
To run the program, use the "make" or "make run" command in terminal. After that, the program will output the result in terminal.
